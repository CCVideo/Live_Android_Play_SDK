1.概述

Android Live SDK 是适用于Android平台的云直播SDK，使用此SDK可以与CC视频云直播服务进行对接，在Android端使用CC视频的直播回放功能。

-keep public class org.webrtc.**{*;}
-keep class com.intel.webrtc.base.**{*;}
-keep interface com.intel.webrtc.base.**{*;}
-keep class com.tencent.smtt.**{*;}

// 设置Surface生命周期监听

也可以分别调用设置播放器、设置文档控件、设置直播参数三个方法完成：

/*****(推荐方案)*****/

dwLiveReplay.setReplayPlayer(player);
// 设置文档控件
dwLiveReplay.setReplayDocView(docView);
// 设置回放参数（回调 及 上下文）
dwLiveReplay.setReplayParams(dwLiveReplayListener, context);

3.10 快速集成说明

完成上面的步骤后，基本上我们就能够观看到直播回放的视频画面和文档画面了，也就基本完成了CC直播回放的核心功能的集成，即观看回放视频和观看回放文档的功能。后续在第4节会对CC直播SDK提供的其他的功能特性的使用做相关说明。

4.功能使用

功能使用时相关的核心类：DWLiveReplayListener（回放回调接口类），DWLiveReplay（回放核心类）。

4.1 聊天功能

回放的聊天功能只包括公共聊天内容，此功能为方法回调功能，相关类DWLiveReplayListener，相关方法如下：

  • 收到回放聊天信息：onChatMessage(TreeSet replayChatMsgs)

4.2 问答功能

回放的问答的功能包括已公开回答问答和已发布的答案，此功能为方法回调功能，相关类DWLiveReplayListener，相关方法如下：

  • 收到回放问答信息：onQuestionAnswer(TreeSet qaMsgs)

4.3 广播功能

回放的广播的功能是直播时发布的广播信息列表，此功能为方法回调功能，相关类DWLiveReplayListener，相关方法如下：

  • 收到广播信息：onBroadCastMessage(ArrayList broadCastMsgList)
